Here're some of the project's best features:

*   Binary to octal
*   Binary to decimal
*   Binary to hexadecimal
*   Octal to binary
*   Octal to decimal
*   Octal to hexadecimal
*   Decimal to binary
*   Decimal to octal
*   Decimal to hexadecimal
*   Hexadecimal to binary
